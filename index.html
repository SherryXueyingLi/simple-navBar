Introduction

  • To help you build a navigation bar by just configure a little.
  • Interface for routing is available.
  • Self designed css is supported
Check the demos around the page.

Quick Start

Import the simple-navBar.js and navBar-menu.css file, since nevbar is depends on jquery, please import it first
<script src="src/lib/jquery.js"></script>
<script src="simple-navBar.js"></script>
<link rel="stylesheet" type="text/css" href="stylesheets/navBar-menu.css"></link>
Init the navBar by following $(youElement).navBar([options Object]): 
Check the options in Configure Section.
$(element).navBar({});

Configure

Currently, all the options are optional, but I suggest at least set the tabs. otherwise you will get a empty navBar.
Properity Name Default Value Description
tabs [] Should be a array of objects, in these objects, the following properties are supported.
  • name: used for identify the tab;
  • label: the text shown on menu, if not set, will be same with name.
  • path: a folder path, if you are using angular.js, it will be useful.
  • viewPath: a folder path, if you are using angular.js, it will be useful.
  • href: used to set the 'href' of the element, if not set, same with name, if in second level, it will be parent's name +'-'+name.
  • sub: should be an array, to save sub tabs if current tab, sub tab's structure same the Tabs.

Example: 

var defaultMenu = [
	{name : "document",label : "Documents",sub : [
		{name : "start",label : "Quick Start"}, 
		{name : "config",label : "Configure",href: "myconfigure"}, 
		{name : "api",label : "APIs"}]},
	{name : "download",label : "Download"} ];
autoClose true If true, the sub tab will closed after user click, like nav bar demo.
arrow true If true, there will be an arrow next to menu title for those who have sub tabs, like nav bar demo
pin true If true, the navigation bar will be pined to window, the position is relative the the parent node.
horizon true If true, the menu will be shown in-line, like nav bar demo.

Javascript API

Set the active by script.

$(element).navBar("active","download");

Get the actived menu by script

$(element).navBar("active");

Adding new context

Add single menu context
$(element).navBar("add", {name: 'about', label:'About'});
Add a list of menu context
Note: Duplicated named object will be ignored.
$(element).navBar("add", [{name:'demo', label:'Demo'},{name: 'about', label:'About'}]);
Add a list of menu context, with parent note indecated. 
Note: If parent node do note exist, will add to first level
$(element).navBar("add", [{name:'demo', label:'Demo'},{name: 'about', label:'About'}]);

Removing context

Remove single menu context
$(element).navBar("remove", 'about');
Remove a list of menu context
Note: If the context already dropped, it will be ignored.
Note: Please click 'Add List' in order to test.
$(element).navBar("remove", ['about', 'demo']);
Remove a list of menu context, with parent note indecated. 
Note: Remove a sub option without indicating the parent name is allowed, but indecating parent name could be more secure
Note: Please click 'Add About & Demo' and 'Add to About' before test.
$(element).navBar("remove", ['about.me', 'demo']);
$(element).navBar("remove", ['me', 'demo']);
